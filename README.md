# Sales Tax Calculator

Free 2026 US sales tax calculator, live at [salestaxcalculatorhq.com](https://salestaxcalculatorhq.com).

- Add tax to a price or work backwards from a tax-included total
- July 1, 2026 state rates plus population-weighted average local rates (Tax Foundation midyear table) for all 50 states and DC
- Editable local rate for exact city/county figures
- 51 state pages with preset rates, example tables, and FAQ schema
- Special cases handled: NOMAD no-tax states, Alaska local-only taxes, CA/UT/VA mandatory statewide local rates, Hawaii GET, NJ urban enterprise zones
- 100% client-side; nothing you enter leaves your browser

## Structure

Static site, no build step. Everything deployable lives in `public/`. The 51 state pages, the sitemap, and the homepage state-links block are generated by `generate_state_pages.py` from the STATES table in `public/script.js` (single source of truth). To regenerate after a rate update: `py generate_state_pages.py`.

## Deploy

Cloudflare Pages, build output directory `public`, custom domain apex + www.
